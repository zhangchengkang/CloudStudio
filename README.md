
# 傻瓜教程

## 如何开始
   1. win+r  输入cmd回车
   2. 输入 ipython notebook 回车
   3. 打开localhost:8888
   4. 选择笔记,开始阅读
   5. 新建自己的目录及文件,要多动手敲!
